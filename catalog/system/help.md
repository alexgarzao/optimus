# Help

## Description

Lists all available Optimus skills with descriptions, usage commands, and when to use each one. Helps users discover what's available and choose the right skill for their situation. Provides situational guidance for common workflows.

## Variables

None — presents the full skill catalog.

## Skill

See the full installable version at [`help/skills/optimus-help/SKILL.md`](../../help/skills/optimus-help/SKILL.md).

## Example

```
/optimus-help — lists all skills with descriptions and usage guidance.
```

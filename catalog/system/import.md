# Import

## Description

Import external task artifacts into optimus format. Discovers existing task files, ring pre-dev artifacts (task specs, subtasks), and converts/links them. Re-runnable — only imports what's new. Never deletes original files.

## Variables

None — automatically discovers task files and ring pre-dev artifacts in the project.

## Skill

See the full installable version at [`import/skills/optimus-import/SKILL.md`](../../import/skills/optimus-import/SKILL.md).

## Example

```
/optimus-import — discovers task files and ring pre-dev artifacts, converts/links to optimus format.
```

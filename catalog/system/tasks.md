# Tasks

## Description

Administrative task management for tasks.md. Create, edit, remove, reorder, and cancel tasks. Manage versions (create, edit, remove, reorder) and move tasks between versions. Validates format, dependencies, and ID uniqueness. Supports advance and demote status operations. Runs on any branch.

## Variables

None — operates on docs/tasks.md interactively.

# Report

## Description

Task status dashboard. Reads tasks.md, computes dependency graph, and presents a comprehensive project status report. Shows progress, active tasks, blocked tasks, ready-to-start tasks, dependency graph, parallelization opportunities, velocity metrics, and workspace health. Read-only.

## Variables

None — automatically parses tasks.md.

## Skill

See the full installable version at [`report/skills/optimus-report/SKILL.md`](../../report/skills/optimus-report/SKILL.md).

## Example

```
/optimus-report — full project status dashboard with dependency graph and velocity.
```
